# Moonstone Lane E-Passport

Keep `moonstone-lane-e-passport.html`, `styles.css`, and `script.js` together in the same folder, then open the HTML file in any modern web browser. It is a mobile-friendly event game page.

## Included

- Randomizes one café and three workshop stops whenever a player generates a passport.
- Lets participants create their own passport with a name and uploaded photo.
- Simulates staff QR-code check-in before a stamp is awarded, including the Instagram-follow stamp.
- Unlocks the gemstone keychain or bracelet lucky-draw reward after all five stamps are collected.
- Uses responsive styling for phones and subtle animations, while respecting a device's reduced-motion setting.

## Before publishing

- Replace the decorative QR square with your real QR code or a link to the Historical Museum's check-in page.
- Replace the sample café and workshop names in the `cafes` and `workshops` lists near the end of the HTML file.
- For real event validation, connect each stamp button to your check-in system instead of allowing manual taps.
